		KELOLA DATA BARANG

Disini saya membuat Website untuk Kelola Data Barang.
Web yang dibuat tidak menggunakan Framework karena pelajaran disekolah hanya sampai PHP native
Website yang saya buat menggunakan aplikasi:
1.Notepad++
2.Xampp 
3.Web browser
Di website Kelola Data Barang ini digunakan untuk Menambahkan Data barang yang bisa diedit/diupdate dan dihapus.
-Tombol Tambah berfungsi untuk menambahkan file Input Data Barang yang berisikan Kode,Nama,Deskripsi,Stok,Harga dan Berat. 
-Tombol Edit untuk mengubah apabila data yang diinputkan salah
-Tombol Hapus untuk  menghapus apabila data yang diinputkan salah.
